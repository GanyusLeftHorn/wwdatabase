Basic Revival Potion Supplies Instantly revives designated Resonator, restoring 100 HP. Can only be used once every 60 seconds. Only effective on your own Resonator in Co-op mode.

[Potion Effect]Slightly stabilize the Resonator's status, assist in consciousness recovery. [Precautions]The interval between taking this type of potion is long, it is not recommended to use it excessively in the short term. [Contraindications]Prohibited to use in combination with the same type of potion or frequency stabilizing type of potion.
